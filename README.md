# README.md — Gestion des Salles
## 1.  Présentation du projet

Gestion des Salles est une application Java permettant la gestion des salles, des utilisateurs et des réservations.
Le projet est conçu selon une architecture modulaire, évolutive et maintenable, avec séparation claire des responsabilités (Model, Repository, Service, Controller).

 ## 2.  Objectifs du projet

Gérer les salles (création, modification, suppression, consultation)

Gérer les utilisateurs

Gérer les réservations de salles avec un statut métier

Appliquer les bonnes pratiques JPA

Fournir une base solide pour une future API REST
